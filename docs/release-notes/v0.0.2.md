# Release v0.0.2

## New Features

### Explicit Kiro1M model mapping (#5)

Added explicit mapping for the `kiro-1m` model to ensure correct routing to the Kiro backend.

## Code Improvements

- Update `golang.org/x/sync` to v0.20.0 (#4)

## Upgrade Instructions

### Homebrew

```bash
brew upgrade kirocc
```

### go install

```bash
go install github.com/d-kuro/kirocc/cmd/kirocc@v0.0.2
```
