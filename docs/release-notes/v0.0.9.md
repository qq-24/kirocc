# Release v0.0.9

## Code Improvements

### Shorten trace_id and session_id in non-debug logs (#30)

`trace_id` and `session_id` are now shortened to 8 characters in info/warn/error logs for improved readability. Full UUIDs are preserved in debug-level logs for detailed diagnostics.

Before:

```
trace_id=b87336e8 session_id=a62d4b66-7f10-45c7-9a8c-c2a7d6ce955c
```

After:

```
# info/warn/error
trace_id=b87336e8 session_id=a62d4b66

# debug
trace_id=b87336e8-... session_id=a62d4b66-7f10-45c7-9a8c-c2a7d6ce955c
```
