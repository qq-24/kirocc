# Release v0.3.0

## New Features

### Add Claude Opus 4.8 mappings pinned to 1M context (#67)

The proxy now recognizes Claude Opus 4.8. `claude-opus-4-8` and `claude-opus-4-8[1m]` are added to the ordered model map, both routed to the Kiro SKU `claude-opus-4.8` and pinned to a 1M context window (no 200k SKU exists upstream, mirroring the 4.6 / 4.7 behavior).

- Resolve and end-to-end response-model behavior is covered by new test cases (bare, `[1m]` exact-match, `context1M`, and Kiro-form input).
- The README mapping table and surrounding prose now include 4.8 alongside 4.6 and 4.7.

## Code Improvements

- Update `modernc.org/sqlite` to v1.51.0 (#66)
- Update `go.opentelemetry.io/contrib/instrumentation/net/http/otelhttp` to v0.69.0 (#65)
- Update opentelemetry-go monorepo to v1.44.0 (#64)
