# Release v0.0.4

## New Features

### Anthropic-Beta context-1m detection (#10)

Detect `context-1m` prefix in the `Anthropic-Beta` header to automatically select the 1M model variant (`-1m` suffix). This replaces the previous `thinkingHint`-based approach and correctly handles Claude Code's 1M context mode.

### Debug header logging (#10)

Add debug-level header logging at all 4 proxy points:

- Client → kirocc (incoming request headers)
- kirocc → Kiro API (outgoing request headers)
- Kiro API → kirocc (upstream response headers)
- kirocc → Client (outgoing response headers)

Uses `slog.LogValuer` for zero-cost when debug logging is disabled. Sensitive headers (`Authorization`, `Proxy-Authorization`, `X-Api-Key`, `Cookie`, `Set-Cookie`, `X-Amz-Security-Token`, `*-Token`, `*-Secret`) are automatically redacted.

### `make debug` target (#10)

Added `make debug` to run kirocc with debug logging enabled.
