# Release v0.0.7

## New Features

### OpenTelemetry Tracing Support (#20)

Added OpenTelemetry tracing instrumentation to the proxy server. This enables distributed tracing for debugging and observability of request flows through the proxy. Traces can be exported to any OpenTelemetry-compatible backend.

### File Logging with Lumberjack Rotation (#21)

Added file-based logging with automatic log rotation powered by lumberjack. Logs can now be written to a file in addition to stderr, with configurable rotation settings (max size, max backups, max age, compression).

## Dependencies

- Update modernc.org/sqlite to v1.46.2 (#19)

### Homebrew
